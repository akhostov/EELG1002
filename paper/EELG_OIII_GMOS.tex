		\section{Data}
		\label{sec:data}
		
		The program GS-2017A-FT-9 (PI: Kiyoto Yabe) is a Fast Turnaround program  that was found within the Gemini Science Archive as part of a larger search of all spectroscopic data that were taken within the COSMOS field ({\color{red} Khostovan et al, in prep ZZZ}). The main focus of the program was to target extremely strong emission line galaxies at $z > 0.3$ that were identified by strong nebular excess in the Subaru/HSC broadband filters. These sources are expected to have strong \oiii~EW of $>2000$\AA~and are metal-poor with the possibility of \oiii4363\AA~detection. 
		
		\subsection{GMOS Spectroscopy}

		GS-2017A-FT-9 consisted of three masks with only one in COSMOS which is the focus of this paper. Observations were taken in 2017 April 23 on Gemini-South using the Gemini Multi-Object Spectrograph (GMOS) with the Hamamatsu Detector, R150 grating, and GG455 blocking filter. This allows for spectral coverage from {\color{red} ZZZ} to {\color{red} ZZZ} with the blocking filter ensuring second-order {\color{red} ZZZ}. Table \ref{table:obslog} shows the observation log identifying the frame ID as found in the Gemini Science Archive, along with the frame type, exposure time, and central wavelength (dispangle). Seeing conditions were reported as $0.9''$ throughout the observing time and humidity levels were somewhat elevated at $\sim 40\%$. A total of 3 individual exposures were pointed at GS2017AFT009-02 (mask design file) with the central wavelengths of each exposure being 8070\AA, 8200\AA, and 8330\AA, respectively, which when 2D Coadded would take into account the GMOS detector chip gaps allowing for continuous spectral coverage in the final 1D and 2D spectra. Each science exposure was 1200 seconds resulting in an on-source integration time of 3600 seconds. Slit widths were set to $0.5''$ for the science targets and box slits of $2''$ sizes were used for alignment stars. Calibration frames consisted of a single 5 second twilight flat taken in the beginning of the night, a single 1 second dome flat (GCALflat) for each dispangle, and a 20 second CuAr lamp exposure for wavelngth calibration per each dispangle. Bias frames were taken on 2017 April 29 (last night of the program) and consisted of 5 frames each 1 second in exposure time.

		
		The standard star LTT7379 (G0 star) that we use for flux calibration purposes was observed on 2017 April 29 with the same configuration and central wavelength of 8200\AA. Conditions were reported with seeing as $1.08''$ and humidity levels of $14\%$. Observations of the standard star were done using a $0.5''$ wide longslit for an on-source integration time of 10 seconds. Associated CuAr lamp and Dome Flat frames were observed in 1 second exposures with the same configuration and slit widths.

		\section{Methodology}
		\subsection{Data Reduction}
	We reduce all raw spectroscopic data using \pypeit~(\citealt{Prochaska2020}; \texttt{v1.10dev} -- \texttt{gemini\_gmos\_mask\_ingestion} branch), a \texttt{python} semi-automated data reduction pipeline that supports many ground- and space-based facilities. The pipeline starts by reading setup files which list the location of the science/standard \& associated calibration frames. These setup files are made available via the linked github repository\footnote{ZZZ GITHUB LINK} for any one to reproduce the final spectra shown in this work. The first main script of the pipeline automatically does bias \& overscan subtraction, slit edge detections, flat fielding, removal of cosmic rays, wavelength calibrations, tilt corrections, sky subtraction, flexure corrections, and a first pass on object extraction using the optimal extraction \cite{Horne1986} and boxcar approach. 
		
		At the time that data was reduced, \pypeit~officially supported the use of slit mask designs for Keck instruments to identify slit edges and object positions along the sit. This was not officially supported for Gemini/GMOS but was available in the \texttt{gemini\_gmos\_mask\_ingestion} development branch which is why we reduced all our data using this specific branch. Currently this has now been merged into the official \pypeit~release versions.
		
		The second step in the pipeline is to 2D coadd all 3 reduced science frames of varying central wavelengths (accounts for GMOS chip gaps in final 2D coadded spectra). To account for potential drift during observations, we define the offsets by using the brightest object in the mask. \pypeit~models the spatial profile of the object and calculates the spatial shift in reference to the first science frame in our 2D coadd list. 		\subsection{Spectroscopic Redshift \& Emission Line Profile Fitting}
		\label{sec:specz_line_profile}
		
		We use \pyqsofit~\citep{Guo2018} for fitting the profiles of all emission lines observed in the GMOS spectra which requires an initial estimate of the spectroscopic redshift visually measured as $z = 0.8275$ using \verb|SpecPro| \citep{Masters2011}. Each detected emission line is fit by a single Gaussian profile with the central wavelength allowed to vary to capture variations in the final measured spectroscopic redshift. Line widths of each emission line are set to be constant among other associated lines within the atomic specie (e.g., Balmer line FWHMs are the same from \hbeta~to H$\eta$). This allows us to deblend lines from different atomic species that are in close wavelength proximity to one another (e.g., \neiii3968\AA~and H$\epsilon$ separated by 2.6\AA; rest-frame) and are not fully resolved given the GMOS wavelength resolution limit. The \oii3726,3729\AA~doublet is also not resolved for which we fit two Gaussians assuming the FWHM from \oiii5007\AA~to deblend the doublet. Line fluxes are measured by integrating the best-fit Gaussian profiles.
		
		%. The center of each Gaussian is allowed to vary up to $\sim 2.7$\AA~allowing for variations in the final measured spectroscopic redshift. Table \ref{table:line_fits} shows the best-fit redshifts about the line centers consistent with $z = 0.8275^{+ZZZ}_{-ZZZ}$.
		
		
		Uncertainty measurements in the emission line profiles are measured via Markov Chain Monte Carlo (MCMC) where \pyqsofit~samples the error spectrum a total of 1000 times and perturbs the 1D spectra accordingly in each realization. We ignore the first 25 samples in the MCMC which is done to ensure that starting points do not affect the underlying probability distribution functions and uncertainty measurements. A thinning of 10 is also incorporated to sample values that are nearly independent from one another. Overall, each emission line detected has an associated probability distribution that will be used throughout this work to measure line ratios and associated uncertainties. 
		
		\subsection{Velocity Dispersion}
		
		The observed velocity dispersion, $\sigma_{obs}$, is measured using the best-fit Gaussian emission line profile. We measure the `intrinsic' velocity dispersion, $\sigma$, by:
		\begin{equation}
			\centering
			\sigma = \sqrt{ \sigma_{obs}^2 - \sigma_{inst}^2 - \sigma_{th}^2 }
		\end{equation}
		which subtracts out the effects of instrumental ($\sigma_{inst}$) and thermal ($\sigma_{th}$) broadening. The latter is measured as $\sigma_{th} = \sqrt{k_B T_e / m_H}$, where $k_B$ is the Boltzmann constant, $m_H$ is the mass of Hydrogen, and $T_e$ is the electron temperature which we use our measurement described in \S\ref{sec:ne_and_te} and Table \ref{table:ISM}. The instrumental broadening is measured using the FWHM of the arc lines ($2.5$ pix with $3.88$ \AA~pix$^{-1}$ resolution).

		\subsubsection{Ionization Parameter}
		
		The ionization parameter is defined as:
		\begin{equation}
			U = \frac{Q_{H^0}}{4 \pi  R_S^2 c n_H}
		\end{equation}
		where $Q_H^0$ is the number of hydrogen ionizing photons per unit time, $R_S$ is the Str\"{o}mgen radius, $c$ is the speed of light, and $n_H$ is the hydrogen number density. Essentially it traces how many ionizing photons are produced within an H{\sc ii} region in comparison to the present number of hydrogen atoms. The higher the ionization parameter would signify more ionizing photons and/or less hydrogen atoms that can be interpreted as energetic ISM conditions caused by: harder ionizing radiation field (\zzz), electron density (\zzz), higher SFR (\zzz), lower metallicities (\zzz), and higher SFR surface densities (e.g., \zzz~NAVEEN). 
		
		The classical diagnostic used to trace $U$ is the O32 ratio given that O$^{++}$ requires 35 eV compared to O$^{+}$ which requires 13.6 eV such that it is sensitive to the ionization state of gas within the H{\sc ii} region (\zzz). Another diagnostic typically used is the Ne3O2 ratio where Ne$^{++}$ requires $\sim 41$ eV and is insensitive to dust attenuation given the $\sim 140$\AA~separation between the two lines \citep{Levesque2014}. However, such diagnostics are limited in range of $U$ and gas-phase metallicities \citep{Kewley2019}. Therefore, we measure ionization parameters using three methods that include photoionization modeling with $U$ as an input:  {\sc HCM} \citep{PerezMontero2014}, {\sc Bagpipes} \citep{Carnall2018}, and {\sc Cigale} \citep{Boquien2019}.
		
		We use {\sc HCM} v5.3 (latest version) which uses {\sc Cloudy} v17 (\zzz Ferland2017) with a fixed $n_e = 100$ cm$^{-3}$ to model the nebular gas. This is coupled with {\sc BPASS} v2.1 (\zzz Eldrige2017) that assumes an instantaneous burst of star formation with an age of 1 Myr, IMF with slope $-1.35$, upper mass limit of $300$ \msol, and includes binary stellar populations. The combination of both forms a grid of modeled line ratios (e.g., \oiii/\oii) as a function of O/H, N/O, and $U$. The observed line fluxes and ratio are then compared to the modeled grid and used to measure the best-fit gas-phase abundances and $U$. We refer the reader to \S\ref{sec:cigale} and \ref{sec:bagpipes} for details on how {\sc Cigale} and {\sc Bagpipes}, respectively, is used to measure $U$ via simultaneously fitting the observed GMOS 1D spectra and multi-wavelength photometry.

		\subsubsection{Bagpipes}
		\label{sec:bagpipes}
	
		{\sc Bagpipes} is a SED suite that allows for the simultaneous fitting of 1D spectra and multi-wavelength photometry as well as parametric and nonparametric star formation history (SFH) modeling. It uses bayesian inference along with nested sampling of the parameter space using {\sc MultiNest} to measure best-fit properties. Table \zzz~highlights the range in parameter space we allow for {\sc Bagpipes} to explore and briefly describe them here. We use the BPASS v2.2.1 SED templates \citep{Stanway2018} which includes the contribution of binary stellar populations. This assumes a broken power law IMF with a $0.1 - 0.5$ \msol~slope of $-1.35$ followed by a steeper IMF slope of $-2.35$ up to an upper mass limit of $300$ \msol. The broken power law shape allows for an increased contribution of older, low-mass stellar populations compared to the \cite{Chabrier2003} IMF which exponentially cuts off at $0.1$ to $1$ \msol. We assume metallicities from 0.0001 to 5 \zsol~that is sampled using a logarithmic prior. The \cite{Calzetti2000} dust attenuation curve is assumed with $A_V$ between 0 to 10 mag and the ratio of nebular-to-stellar reddening set to unity. Templates for the nebular component is recomputed using the BPASS stellar grid and {\sc Cloudy v17.03} to cover the ionization parameter range of $\log_{10} U = -4$ to $-1$ in order to consider cases of extreme energetic ISM conditions. 
		
		We use the \cite{Leja2019} continuity non-parametric SFH implementation in {\sc Bagpipes} to better understand how past star-formation activity could have occurred in EELG1002 based on the available spectrophotometric data. The time bins assumed in the SFH modeling starts with short duration and increases with each successive bin: 0, 1.5, 3, 10, 30, 100, 300, 1000, 3000, and 6000 Myr. The shorter time scales are to gauge how rapidly the recent burst of star formation formed and is constrained primarily by the multiple strong Balmer line detections that directly trace instantaneous star formation activity. The longer timescales are primarily constrained by photometry that well constrains the stellar continuum and is sensitive to past star-formation activity that occurred at older lookback times.

	\subsection{Measuring Equivalent Width}
	
	Equivalent width (EW) is measured as the ratio between emission line flux and its associated continuum flux density. We measure the emission line flux using the best-fit emission line profile as described in \S\ref{sec:specz_line_profile}. The continuum flux density is drawn from the best-fit SEDs as measured by both {\sc Cigale} and {\sc Bagpipes}. Although a stellar-nebular reddening ratio of order unity is assumed in the SED fitting process, we note that Balmer Decrements suggest an $E(B-V)_{nebular} \sim 0$ mag while both {\sc Cigale} and {\sc Bagpipes} suggest slight stellar reddening. To account for this attenuation difference, we only correct the best-fit observed continuum flux densities using the {\sc Cigale}-measured $E(B - V)_{stellar}$ along with a \cite{Calzetti2000} dust attenuation curve and compute the equivalent widths for each line. We do not make any dust correction for the nebular emission given that our Balmer Decrement measurements show $E(B - V)_{nebular} \sim 0$ mag.
	
	\subsection{Ionizing Photon Production Efficiency}
	
	The ionizing photon production efficiency, \xiion, traces how well a galaxy can produce ionizing photons (e.g., $> 13.6$ eV that can ionize H{\sc i}) and is defined as:
	\begin{eqnarray}
		\textrm{\xiion} = \frac{N(H^0)}{L_{UV}^{int}}
	\end{eqnarray}
	where $N(H^0)$ represents the production rate of hydrogen ionizing photons and $L_{UV}^{int}$ is the intrinsic, rest-frame, dust-corrected $1500$\AA~continuum luminosity. The definition of $\xi_{ion}$ is slightly varied where we refer the reader to \S3.2 of \cite{Chevallard2018} for a detailed discussion. Some studies use the monochromatic UV luminosity attributed only to the stellar continuum (typically cited as $\xi_{ion}^\star$) and ignores the emission and absorption caused by neutral and ionized gas. Other studies incorporate the nebular and stellar continuum in measuring the $1500$\AA~monochromatic luminosity ($L_{UV}^\textrm{H{\sc ii}}$) and measure the ionizing photon production efficiency (\xiionHtwo). We adopt this definition throughout this paper using \xiion~as our defined notation for the nebular + stellar continuum contributions.
	
	We use the best-fit combined stellar and nebular continuum fits from {\sc Cigale} to measure $L_{UV}$ using a tophat filter centered at $1500$\AA~with a width of 100\AA. We use the \cite{Leitherer1995} calibration:
	\begin{eqnarray}
		N(H^0) = 2.1\times10^{12} L_{int}(\textrm{H}\beta)~\textrm{[s$^{-1}$]}
	\end{eqnarray}
	to measure the ionizing photon production rate, $N(H^0)$, with dust-corrected \hbeta~luminosity from our observed GMOS spectra. Given that Balmer Decrements show $E(B-V)_{nebular} \sim 0$ mag, we do not apply any dust correction to the \hbeta~luminosity. However, our best-fit {\sc Cigale} measurements find $E(B-V)_{stellar} \sim$ \zzz~mag such that we need to apply a dust-correction to our $L_{UV}$ measurement. This is done by assuming a \cite{Calzetti2000} dust attenuation curve with $A_{1500\textrm{\AA}} = $ \zzz~mag.

		\subsection{Cause of High Electron Temperatures and  Ionization Parameter}
		
		Could be due to a multitude of combined factors. Metal coolants are needed to absorb ionizing photons that heat up the surrounding gas. In the case of low metallicity, there is not enough absorbants to cool the system. Couple that with low dust, this allows for gas heating and ionizing photon escape into the ISM. Indeed, this galaxy also has an inferred Lyman Continuum $f_{esc}$ of 10\% which would also decrease the ionizing parameter $\log_{10} U$.
